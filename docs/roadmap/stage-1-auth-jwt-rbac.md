# 阶段 1：注册登录、JWT 与 RBAC

返回主路线图：[活动预约与票务平台建议路线图](./event-booking-roadmap.md)

## 目标

完成用户注册、登录、获取当前用户、基于角色的权限控制。

## 范围

- User/Role/UserRole 数据模型。
- 密码加密。
- JWT 鉴权。
- Spring Security 基础接入。
- 管理员种子数据。
- 基础权限注解与鉴权流程。

## 非范围

- OAuth。
- 短信/邮件验证码。
- 找回密码。
- 复杂权限中心。

## 要求

- 明确表结构。
- 明确 token 策略。
- 明确异常场景。
- 明确权限边界。
- 明确测试点。
